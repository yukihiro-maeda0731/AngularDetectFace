# AngularDetectFace

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.11.

## アプリの概要
お出かけ前に使用することを想定した笑顔判定アプリです。
HTMLVideoで撮影した画像をAWSRekognitionで笑顔かどうか判定し、OKであれば「いってらっしゃい!」NGであれば「もう一度笑顔で！」と
表示します。
